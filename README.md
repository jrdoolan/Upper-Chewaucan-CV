# Upper Chewaucan Watershed Restoration — Contingent Valuation Project

**Author:** Joseph Doolan | **Course:** EC434 | **Date:** November 2024

---

## Overview

This project estimates public Willingness to Pay (WTP) for the Upper Chewaucan Watershed Restoration in Lake County, Oregon. Using a contingent valuation (CV) survey and logit regression (Hanemann 1984), we estimate the average annual WTP as an additional state tax.

The project is led by the Lake County Umbrella Watershed Council (LCUWC) in partnership with ODFW and the Oregon Watershed Enhancement Board.

## How to Run

1. Clone the repo
2. Add `data/survey_responses.xlsx` (the raw Google Form export)
3. Open `analysis/analysis.R` in RStudio and run — figures save to `figures/`

**R packages required** (auto-installed via `pacman`):
`tidyverse`, `ggplot2`, `readxl`, `dplyr`, `modelsummary`, `mfx`, `stargazer`

## Notes
